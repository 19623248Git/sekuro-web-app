# Overview

This is the repository for Sekuro 18 (2026) Web Application. Sekuro is an orientation for the robotics organization that spans across multiple weeks. This website serves to give important information about the orientation's timeline of events and aims to also be a hub for all important links for sekuro participants.

## Clone and run locally

1. Clone the repository

```bash
# git clone
git clone https://github.com/19623248Git/sekuro-web-app.git

# gh cli
gh repo clone 19623248Git/sekuro-web-app
```

2. Create a new file `.env.local` and replace it with vercel's environment variables (contact the owner of this repository)

3. Install nodejs dependencies

```bash
npm install
```

4. Run the application in `dev` mode

```bash
npm run dev
```
